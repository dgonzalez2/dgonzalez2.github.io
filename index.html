My Portfolio

The Crimson Dungeon

Developed as the culmination of my university studies. Developed entirely on my own from code to the graphics. Using procedurally generated levels for infinite replayability. It is a Rouge-Like, FPS, Dungeon Crawler, Sci-Fi game set in a place called The Crimson Dungeon. Where the player must fight against enemies and find a portal to the next floor. It also features gun customization and different upgrades for both the player and weapon.

    Royale Combat

    This game was made for the Inter Game Jam with a group of 5. This is a card based strategy game where the player must summon minions to fight the other player's minions and eventually kill the other player. The Player also takes an active role as an avatar in the battlefield where they must run awway from the minions to prevent getting attacked by them. I was in charge of dealing with the AI of the minions and there respective behaviours for the different types of minions.

    Grimm: Sugar Rush Frequency

    Made for the Reload Magazine Game Jame #1, this was my first published Unity title. The theme for the jam was "Fairy Tale", and so I concocted a game based around Gretel from Hansel and Gretel. The game is first person shooter with a focus on movement as the player is equipped with a grappling hook to use across the levels. I developed the game on my own as such I did all the art and program necessary for it to run. Due to the movement implementations I learned a lot about using physics manipulating it to make the grappling be a fun mechanic to use and traverse the area. The art direction I took with this project was inspired by American McGee's Alice which you can see in the enemy and weapon design. This project overall still remains as one of my favourites to have made and hopefully revisit.
